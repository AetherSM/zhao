# 数据库设计文档（MySQL 8.0）

## 1. 基本信息

- **数据库名**：`tutoring_manage_system`
- **字符集**：utf8mb4
- **排序规则**：utf8mb4_unicode_ci
- **初始化脚本**：`src/main/resources/sql/schema.sql`

## 2. 表清单

### 2.1 系统表（权限/账号）

- `sys_user`：用户表
- `sys_role`：角色表
- `sys_user_role`：用户-角色关联表（多对多）

### 2.2 业务表（辅导机构核心业务）

- `student`：学员
- `teacher`：教师
- `course_category`：课程分类
- `course`：课程
- `student_course`：报名（学员-课程）
- `course_schedule`：排课
- `payment`：缴费记录
- `student_attendance`：学员考勤

## 3. 表结构（字段字典摘要）

> 详细建表以 `schema.sql` 为准；此处给出“论文/说明书友好”的字段说明。

### 3.1 `sys_user`（用户表）

- **主键**：`id` bigint
- **核心字段**：
  - `username`：账号（唯一）
  - `password`：密码（演示脚本为明文；建议生产使用 BCrypt）
  - `real_name`：姓名
  - `status`：状态（0 禁用 / 1 正常）
  - `deleted`：逻辑删除（0/1）
  - `create_time` / `update_time`：审计字段

### 3.2 `sys_role`（角色表）

- `role_code`：角色编码（如 `SUPER_ADMIN` / `ORG_ADMIN` / `TEACHER` / `FINANCE`）

### 3.3 `sys_user_role`（用户角色关联）

- **唯一约束**：同一用户同一角色仅一条（`uk_user_role(user_id, role_id)`）

### 3.4 `student`（学员）

- `student_name`、`gender`、`grade`、`enroll_time` 为核心展示字段
- `deleted` 为逻辑删除（配合 MyBatis-Plus）

### 3.5 `teacher`（教师）

- `teacher_name`、`subject`（擅长科目）为核心字段

### 3.6 `course_category` / `course`

- `course.category_id` 指向分类
- 课程总价业务口径：`price * total_hours`

### 3.7 `student_course`（报名）

- **唯一约束**：`uk_student_course(student_id, course_id)` 防止重复报名

### 3.8 `course_schedule`（排课）

- 关键字段：`teacher_id`、`classroom`、`schedule_date`、`class_period`
- **冲突约束**（通过唯一索引保障同一时间段不重复）：
  - `uk_teacher_time(teacher_id, schedule_date, class_period, deleted)`
  - `uk_classroom_time(classroom, schedule_date, class_period, deleted)`

### 3.9 `payment`（缴费记录）

- `amount`、`pay_time`、`pay_type` 为核心字段

### 3.10 `student_attendance`（考勤）

- `attendance_status`：1 正常 / 2 迟到 / 3 早退 / 4 缺勤

## 4. 业务计算口径

### 4.1 欠费统计

- 应缴：`course.price * course.total_hours`
- 已缴：`SUM(payment.amount)`（按 `student_id + course_id` 汇总）
- 欠费：`应缴 - 已缴`（>0 则为欠费）

## 5. 表关系说明（ER 关系）

- `sys_user` ← 多对多 → `sys_role`（通过 `sys_user_role`）
- `student` ← 多对多 → `course`（通过 `student_course`）
- `course` → `course_category`（多对一）
- `course_schedule` → `course`（多对一）
- `course_schedule` → `teacher`（多对一）
- `payment` → `student`（多对一）
- `payment` → `course`（多对一）
- `student_attendance` → `student`（多对一）
- `student_attendance` → `course_schedule`（多对一）
