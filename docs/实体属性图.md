## 1) ER 图（表关系）

```mermaid
erDiagram
  SYS_USER ||--o{ SYS_USER_ROLE : has
  SYS_ROLE ||--o{ SYS_USER_ROLE : has

  COURSE_CATEGORY ||--o{ COURSE : contains
  STUDENT ||--o{ STUDENT_COURSE : enrolls
  COURSE ||--o{ STUDENT_COURSE : enrolled_by

  TEACHER ||--o{ COURSE_SCHEDULE : teaches
  COURSE ||--o{ COURSE_SCHEDULE : scheduled

  STUDENT ||--o{ PAYMENT : pays
  COURSE ||--o{ PAYMENT : paid_for

  STUDENT ||--o{ STUDENT_ATTENDANCE : attends
  COURSE_SCHEDULE ||--o{ STUDENT_ATTENDANCE : attendance_of

    varchar password

    bigint user_id
    bigint role_id
  }

    bigint id PK

    int total_hours
  }
  STUDENT_COURSE {
    bigint id PK

  }

    bigint id PK
    bigint course_id FK
    bigint teacher_id FK

  }
  PAYMENT {
    bigint id PK

  }
  STUDENT_ATTENDANCE {
    bigint id PK

  }
```
