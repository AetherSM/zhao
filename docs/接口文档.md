# 接口文档（API v1）

## 统一规范

- 前缀：`/api/v1`
- Header：`Authorization: Bearer <token>`（除登录接口外）
- 统一返回：

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```

## 1. 用户登录

### POST `/api/v1/user/login`

请求体：

```json
{"username":"admin","password":"123456"}
```

响应 `data`：

```json
{
  "token": "xxxx",
  "tokenType": "Bearer",
  "userId": 1,
  "username": "admin",
  "roles": ["SUPER_ADMIN"]
}
```

## 2. 学员管理

### GET `/api/v1/student/list`

Query：

- `page` 默认 1
- `size` 默认 10
- `studentName`（可选，模糊）
- `grade`（可选，精确）
- `phone`（可选，模糊）

### POST `/api/v1/student`

请求体示例：

```json
{"studentName":"王小宝","gender":1,"age":9,"grade":"小学 3 年级","phone":"13900139000","enrollTime":"2024-01-10"}
```

### PUT `/api/v1/student/{id}`

### DELETE `/api/v1/student/{id}`

## 3. 排课管理（含冲突检测亮点）

### GET `/api/v1/schedule/list`

Query：

- `teacherId`（可选）
- `courseId`（可选）
- `scheduleDate`（可选，`yyyy-MM-dd`）

### POST `/api/v1/schedule`

请求体示例：

```json
{"courseId":2,"teacherId":1,"classroom":"101 教室","scheduleDate":"2024-03-15","classPeriod":"上午 1-2 节"}
```

### GET `/api/v1/schedule/checkConflict`

Query：

- `teacherId`
- `classroom`
- `scheduleDate`
- `classPeriod`

返回：

- `true`：有冲突（同一教师或同一教室同一时间已排课）
- `false`：无冲突

### DELETE `/api/v1/schedule/{id}`

## 4. 缴费管理（含欠费统计亮点）

### GET `/api/v1/payment/list`

Query：

- `studentId`（可选）
- `courseId`（可选）
- `payTimeStart`（可选，ISO 时间：`2024-03-10T10:00:00`）
- `payTimeEnd`（可选）

### POST `/api/v1/payment`

请求体示例：

```json
{"studentId":1,"courseId":2,"amount":3200.00,"payTime":"2024-03-10T10:00:00","payType":"微信","remark":"全额缴费"}
```

### GET `/api/v1/payment/arrears`

说明：根据“报名表 student_course + 课程总价（price*total_hours）- 已缴费用 SUM(payment.amount)”计算欠费。

Query：

- `studentName`（可选，模糊）

返回字段：

- `totalFee`：应缴总额
- `paidFee`：已缴总额
- `arrearsFee`：欠费
